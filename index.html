NotebookLM Interactive Audio Agent (Google)

This work is part of Google's NotebookLM product. Try it here

The core challenge was enabling users to interrupt AI responses mid-stream while maintaining low latency and consistent system behavior. Unlike traditional request/response systems, this required coordinating streaming audio output, user input, and AI generation pipelines in real time.

To support this, I built a node-based orchestration system that allowed components in the pipeline to be dynamically swapped without breaking dependencies. This enabled rapid iteration in a fast-moving research environment while maintaining production stability at scale.

Key design considerations:

  • Interruptible streaming: handling mid-response cancellation and recovery
  • Low-latency pipelines: minimizing delay across audio, network, and AI processing
  • Dynamic composition: enabling flexible agent behavior without redeploying systems
  • Scalability: supporting large numbers of concurrent real-time sessions

App Safety - Android Review Platform

I was the team lead for the review platform used by 2,400+ human agents reviewing over 1M monthly app reviews. While supporting these users, without interruption, I led the in-place migration from GWT (Java) to Angular with no interruption of service. In order to do this (with a team of contractors), I enforced a rigid separation of model logic & view logic. All model logic remained in Java, but was transpiled to ClosureJS with a custom communication layer to the view logic. This made it so we could preserve all business logic & existing unit tests and focus only on converting the UI implementation to Angular.

Impact: Enabled seamless migration of a critical system with zero downtime while increasing development velocity.

Medical Imaging (Google Brain)

I led the Frontend team for Google Brain's pathology research. To train AI models for cancer research, we needed to acquire huge amounts of training data from experts in the field. To do this, I designed and built a imaging platform with annotation tools used by contracted Pathologists across the country. This platform's imaging is like a Google Maps for tissue samples, allowing the user to pan/zoom and annotate cells and regions.

Github Repo
